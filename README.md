## Installation

```bash
$ git clone https://github.com/BalanaguYashwanth/price-chain-tracking

$ yarn install
```

## Running the app

```bash
# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Docker
```bash
$ docker pull byashwanth/nestjs_price_tracking:latest

$ docker compose up
```
## Swagger

